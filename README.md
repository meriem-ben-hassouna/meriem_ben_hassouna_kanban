Nom du projet : Tableau Kanban

Description : Ce projet est un site web permettant de s'organiser grace a un tableau Kanban. On peut y créer des catégories (max 5) et attribuer une catégories a chaque tâche qu'on peut ensuite deplacer dans le tableau selon le progrès du travail ( To do, In progress, Done).

Technologies utilisées :
  -HTML : structure du site
  -CSS : mise en forme et responsive design
  -JavaScript : interactions, gestion des données
  -LocalStorage : sauvegarde des informations côté navigateur
  -Git & GitHub : versioning et hébergement du projet
  -GitHub Pages : mise en ligne du rendu final

Fonctionnalités principales :
  -Gestion des catégories
  -Création de cartes
  -Ajout de tâches
  -Système de drag & drop
  -Validation des formulaires
  -Sauvegarde automatique des données (Local Storage)
  -Restauration des informations sauvegardées
  -Interface simple et responsive

Lien vers Github pages : rendu final : https://meriem-ben-hassouna.github.io/meriem_ben_hassouna_kanban/

Nouveautés explorées :
  -Organisation d’un projet web complet (HTML/CSS/JS)
  -Utilisation de LocalStorage pour stocker des données 
  -Gestion des événements (click, dragstart, drop …)
  -Utilisation de GitHub Pages 

Difficultés rencontrées :
  -Alignement et espacement corrects en CSS
  -Gestion de la sauvegarde/restauration via LocalStorage
  -Problèmes dans l'installation de Git : Git n’était pas reconnu dans le terminal.
  -PATH non configuré : la commande git ne fonctionnait pas parce que Git n’était pas ajouté au PATH.
  -Erreur dans la création des catégories : les catégories ne s’ajoutaient pas correctement ou ne s'affichaient pas dans l'interface.

Solutions apportées :
  -Tests répétés pour comprendre le comportement du DOM
  -Restructuration du CSS (suppression de marges inutiles, spacing ...)
  -Revue du code JS pour le rendre plus clair et organiser les fonctions
  -Ajout manuel du PATH de Git dans les variables d’environnement pour permettre l’utilisation des commandes git dans le terminal.
  -Pour les catégories : j’ai demandé de l’aide à ChatGPT, ce qui m’a permis de comprendre l’erreur dans ma fonction et de corriger la logique JavaScript.
